"# AI Study Planner\n\n## Onboarding Guide\n\n1. **Setup Environment**\n   - Install Node.js (v18+)\n   - Install pnpm (v8+)\n\n2. **Install Dependencies**\n   - Run `pnpm install`\n\n3. **Database Setup**\n   - Run `wrangler login`\n   - Run `wrangler d1 create ai-study-planner`\n   - Run `wrangler d1 execute ai-study-planner --remote --file=schema.sql`\n\n4. **Development Setup**\n   - Run `pnpm dev` to start the development server\n\n5. **P620 Setup**\n   - Configure Windows Task Scheduler for 6am runs\n   - Set up SMTP for 6am text notifications\n\n6. **Deployment**\n   - Deploy with `wrangler deploy`\n"
